## 1. 概述

### 1.1 背景

BEST（全超导托克马克）是一台先进的全超导托克马克核聚变实验装置，其主机系统包括磁体系统（纵场、极向场、中心螺线管、校正场磁体）、双层壳体真空室、主机杜瓦、冷屏系统等核心部件。

电子回旋共振加热（ECRH）是 BEST 装置的核心辅助加热手段之一。**与仿星器（如 W7-X）不同，托克马克中 ECRH 系统承担多重关键功能**：

| 功能 | 说明 | 托克马克特有 |
|------|------|:----------:|
| **等离子体加热** | 直接加热电子，提升等离子体温度 | |
| **电子回旋电流驱动（ECCD）** | 驱动局部等离子体电流，优化电流分布 | ★ |
| **NTM 抑制** | 在新经典撕裂模（NTM）磁岛位置精确注入，抑制不稳定性 | ★ |
| **锯齿控制** | 通过局部加热/电流驱动控制锯齿不稳定性 | ★ |
| **等离子体击穿辅助** | 预电离辅助等离子体启动 | ★ |
| **逃逸电子抑制** | 破裂后利用 ECRH 抑制逃逸电子束 | ★ |
| **高密度运行** | 参考 W7-X 经验，O2 模式多程吸收方案 | |

本方案以 W7-X ECRH 系统的成功设计经验为参考蓝本（特别是其 10 MW 升级方案），**针对 BEST 托克马克的装置结构特点和物理需求**，设计一套完整的 10 MW 级 ECRH 控制保护系统。

### 1.2 与 W7-X 设计的关键差异

| 对比项 | W7-X（仿星器） | BEST（托克马克） | 设计影响 |
|-------|---------------|-----------------|---------|
| 磁场拓扑 | 三维非轴对称 | 轴对称 | 端口布局不同，微波注入几何不同 |
| 等离子体电流 | 无 | 有（MA级） | 新增破裂保护、ECCD 功能 |
| 破裂风险 | 无 | 有（重大风险） | 新增快速关断、launcher 保护 |
| VDE 风险 | 无 | 有 | launcher 需承受电磁力 |
| 主要加热功能 | 纯加热 | 加热 + 电流驱动 + NTM 抑制 | 需实时束调向，精度更高 |
| 外围结构 | 无杜瓦 | 杜瓦 + 冷屏 | 传输线需穿越多层结构 |
| 超导磁体 | 仅主场线圈 | TF+PF+CS+校正场（全超导） | 杂散磁场约束更严格 |
| 核辐射环境 | 弱 | 较强（D-D/D-T 中子） | 近机部件需抗辐射设计 |
| 运行模式 | 稳态为主 | 脉冲/长脉冲/稳态 | 需适配多种放电模式 |

### 1.3 设计原则

| 设计原则 | 说明 |
|---------|------|
| **安全优先** | 充分考虑托克马克破裂、VDE 等特有风险，保护设计满足最严格工况 |
| **高可靠性** | 采用冗余设计，关键保护环节使用硬件联锁，确保设备和人员安全 |
| **模块化** | 各回旋管通道独立控制，可单独运行、维护和升级 |
| **实时性** | 控制环路响应 < 1 ms；NTM 抑制实时追踪 < 5 ms；保护动作 < 100 μs |
| **精确性** | NTM 抑制要求微波束沉积位置精度 < 2 cm（归一化极向磁通 Δρ < 0.05） |
| **灵活性** | 支持 X2/O2 模式切换、ECCD/ECRH 切换、功率和注入角实时调制 |
| **可扩展性** | 预留通道扩展能力（12→16 通道），支持未来升级 |
| **环境适应** | 控制与电子设备远离强磁场和辐射区域，满足 EMC 要求 |
| **接口兼容** | 与 BEST CCS/PCS/MPS 以及磁体、低温、真空等主机系统无缝集成 |

### 1.4 设计依据

- W7-X OP1.2 运行经验：纯 O2-ECRH 在密度 1.4×10²⁰ m⁻³ 下实现稳态运行
- W7-X OP2 升级目标：10 MW，12 条束线，单管功率 1.0-1.5 MW
- W7-X 多程吸收方案：O2 模式多程吸收 > 90%（优化后 95%）
- W7-X 传输线效率：准光学传输约 40 m，效率 95%
- W7-X 空气干燥系统：相对湿度 < 20%（20°C），防止传输线打火
- BEST 装置总体设计文件（磁体、真空室、杜瓦、冷屏参数）

### 2.3 接口需求（ECRH 系统向外部系统提出的需求）

| I-07 | **TF 磁体间隙路由空间** | 磁体系统 | ECRH 传输线需从 TF 线圈间隙穿过，需确认间隙尺寸和可用空间 | 磁体设计方提供空间信息 |

### 2.4 关键边界澄清

以下几处边界在设计过程中容易混淆，特此明确说明：

#### 2.4.1 NTM 抑制功能的职责划分

NTM（新经典撕裂模）抑制是 ECRH 在 BEST 托克马克中的核心先进功能，涉及多个系统协作。职责划分如下：

```
┌─────────────────────────────────────────────────────────────────────┐
│                       NTM 抑制功能链                                │
│                                                                     │
│  ┌──────────┐   ┌──────────┐   ┌─────────────┐   ┌──────────────┐ │
│  │ 诊断系统  │──→│  PCS     │──→│ ECRH-MC     │──→│ Launcher     │ │
│  │          │   │          │   │             │   │ + 回旋管     │ │
│  └──────────┘   └──────────┘   └─────────────┘   └──────────────┘ │
│                                                                     │
│  │← 诊断系统供货 →│← PCS 供货 →│← ECRH 系统供货 ──────────────── →│ │
│                                                                     │
│  功能分工:                                                          │
│  ─────────                                                          │
│  诊断系统:  采集 ECE 电子温度剖面、Mirnov 磁场扰动信号               │
│            → 不属于 ECRH 供货范围                                    │
│                                                                     │
│  PCS:       接收诊断数据，运行 NTM 检测算法                          │
│            → 输出磁岛位置 (ρ_pol) 和宽度 (w)                        │
│            → 输出目标微波沉积位置 (ρ_dep_target)                     │
│            → 输出各通道功率设定值                                     │
│            → 不属于 ECRH 供货范围                                    │
│                                                                     │
│  ECRH-MC:   接收 PCS 给出的目标沉积位置和功率设定                    │
│            → 运行射线追踪快速算法，将沉积位置转换为 Launcher 角度      │
│            → 下发 Launcher 角度指令和功率指令                        │
│            ★ 属于 ECRH 供货范围                                      │
│                                                                     │
│  Launcher:  执行角度指令，调节微波束注入方向                          │
│            ★ 属于 ECRH 供货范围                                      │
│                                                                     │
│  回旋管:    执行功率指令                                              │
│            ★ 属于 ECRH 供货范围                                      │
└─────────────────────────────────────────────────────────────────────┘
```

**核心原则**：ECRH 系统 **不负责** NTM 磁岛的检测和定位，这属于 PCS 和诊断系统的职责。ECRH 系统 **负责** 将 PCS 给出的目标沉积位置翻译为 Launcher 角度指令并精确执行。

### 2.2 ECRH 系统必须面对的装置约束

#### 约束 1：多层穿越

ECRH 微波束从回旋管大厅到达等离子体，必须依次穿越以下结构：

```
回旋管大厅 → [建筑墙体] → [杜瓦壁] → [杜瓦冷屏] → [TF磁体间隙] 
           → [真空室冷屏] → [真空室双层壁] → [包层/第一壁] → 等离子体

每一层穿越都需要专用的穿越件(penetration)设计
```

| 穿越层 | 关键约束 | ECRH 设计影响 |
|-------|---------|--------------|
| **杜瓦壁** | 真空密封（杜瓦内维持绝热真空） | 需要杜瓦法兰和密封波纹管 |
| **杜瓦冷屏** (~80K) | 不能引入过多热负荷 | 传输线穿越处需热截断设计 |
| **TF 磁体间隙** | 空间极其有限，强磁场(数T) | 束线必须从 TF 磁体间隙穿过，限制束线数量和尺寸 |
| **真空室冷屏** (~80K) | 热屏蔽完整性 | 穿越孔径需最小化 |
| **真空室壁**（双层） | 结构完整性、真空密封、核屏蔽 | 金刚石窗安装在真空室法兰上 |
| **包层/第一壁** | 高热负荷、中子辐照 | launcher 嵌入包层开口中 |

#### 约束 2：杂散磁场

BEST 全超导磁体系统产生的杂散磁场对回旋管运行构成直接威胁：

| 位置 | 典型杂散磁场强度 | 回旋管要求 | 对策 |
|------|----------------|-----------|------|
| 真空室壁 | 数 T | -- | -- |
| 杜瓦外壁附近 | 50-200 mT | -- | -- |
| 回旋管大厅（距装置 ~20m） | 1-10 mT | **< 0.5 mT**（除回旋管自身超导磁体外） | 距离隔离 + 主动补偿线圈 |
| 回旋管本体 | -- | 自身超导磁体 ~5.5 T | 杂散场影响束流品质 |

#### 约束 3：核辐射环境

| 区域 | 辐射水平 | ECRH 部件要求 |
|------|---------|--------------|
| 真空室内（运行时） | 高（中子通量 ~10¹⁴ n/cm²/s） | launcher 反射镜需耐辐照材料 |
| 真空室外/杜瓦内 | 中等 | 传输线部件需选用抗辐照材料 |
| 端口区域 | 中等 | 金刚石窗需评估辐照损伤 |
| 回旋管大厅 | 低（生物屏蔽后） | 电子设备可正常工作 |

#### 约束 4：端口资源竞争

BEST 真空室端口需在多个系统间分配（诊断、加热、抽气、维护等），ECRH 系统需要争取足够的端口资源：

| 端口类型 | ECRH 需求 | 竞争系统 |
|---------|----------|---------|
| 赤道面端口 | 赤道面发射天线（on-axis ECCD） | NBI、诊断、维护 |
| 上端口 | 上发射天线（off-axis NTM 抑制） | 诊断、弹丸注入 |
| 下端口 | 备选 | 偏滤器抽气、诊断 |

#### 约束 5：托克马克特有事件

| 事件 | 特征 | 对 ECRH 的影响 |
|------|------|---------------|
| **大破裂 (Major Disruption)** | 等离子体电流在 ~1-10 ms 内丧失，释放 ~MJ 级能量 | launcher 承受热冲击和电磁力；需快速关断 ECRH |
| **垂直位移事件 (VDE)** | 等离子体垂直失稳，电流通道撞壁 | launcher 结构承受巨大电磁力 |
| **逃逸电子束** | 破裂后产生的高能(~MeV)逃逸电子束 | 可能损伤 launcher 和第一壁；ECRH 可辅助抑制 |
| **ELM (边缘局域模)** | 边缘周期性能量释放 | 密度和温度扰动影响微波吸收 |
| **NTM (新经典撕裂模)** | 磁岛生长，降低约束性能 | ECRH/ECCD 是主要抑制手段 |

## 8. 传输线子系统控制

### 7.1 传输线布局（BEST 适配）

与 W7-X 的纯准光学传输不同，BEST 方案采用 **准光学 + 波导管混合传输方案**，以适应杜瓦穿越要求：

```
回旋管输出 → MOU → 极化器组 → 多束合束段 → 准光学传输(大气,~25m)
→ 生物屏蔽墙穿越 → 准光学/波导切换 → 杜瓦穿越(波导管,~3m)
→ 杜瓦冷屏穿越 → TF磁体间隙穿越 → 真空室冷屏穿越
→ 金刚石真空窗(真空室端口法兰) → 真空室内 → EC Launcher
```

**与 W7-X 方案的关键差异**：

| 对比项 | W7-X | BEST | 原因 |
|--------|------|------|------|
| 传输方式 | 纯准光学 | **准光学 + 波导管混合** | 杜瓦穿越空间受限，TF间隙小 |
| 穿越层数 | 1层（真空室壁） | **5层**（屏蔽墙+杜瓦+杜瓦冷屏+VV冷屏+VV） | BEST 多层围护结构 |
| 总传输距离 | ~40 m | **~35-40 m** | 取决于回旋管大厅位置 |
| 总传输效率 | 95% | **≥ 92%** | 穿越损耗增加 |
| 特殊需求 | 无 | **热截断设计（冷屏穿越）** | 保护 80K 冷屏 |

### 7.2 环境控制

**W7-X 教训**：2018 年夏季运行期间，由于空气湿度过高导致传输线打火，总功率被迫从 8 MW 降额至 6 MW。

| 控制参数 | 指标 | 控制措施 |
|---------|------|---------|
| 相对湿度（传输线走廊） | **< 20% RH @ 20°C** | 大功率空气干燥系统 |
| 温度（传输线走廊） | 20 ± 2°C | HVAC 系统 |
| 洁净度 | ISO 8 级 | 过滤通风 |
| 湿度监测 | 多点分布式 | 每个传输段设湿度传感器 |
| **杜瓦穿越区温度** | **监测** | **确认热截断有效** |

### 7.3 传输线监控参数

| 监控项 | 传感器 | 报警阈值 | 联锁动作 |
|-------|--------|---------|---------|

| 反射镜冷却水温度 | 温度传感器 | > 50°C | 降功率 |
| 金刚石窗片温度 | 红外温度计 | > 150°C 报警 / > 250°C 联锁 | 关断该通道 |
| 金刚石窗弧光 | 光电探测器 | 弧光检测 | 立即关断（< 100 μs） |
| 传输线区域湿度 | 湿度传感器 | > 20% RH 报警；> 30% 联锁 | 降功率/禁止启动 |

## 9. 天线发射子系统控制（EC Launcher）

### 8.1 EC Launcher 设计概况

**EC Launcher 是 BEST 装置中 ECRH 系统与等离子体的直接接口**，位于真空室内，嵌入包层系统开口中。相比 W7-X 的简单反射镜，BEST 的 launcher 需要满足远更严苛的要求：

| 需求 | W7-X launcher | BEST launcher |
|------|---------------|---------------|
| 物理环境 | 温和（无中子） | **强辐射（中子 + γ射线）** |
| 电磁力 | 无 | **破裂/VDE 时承受巨大电磁力** |
| 热负荷 | 中等 | **高热负荷（等离子体辐射 + 微波）** |
| 调节精度 | ±0.1° | **±0.1°（NTM 抑制需求更严格）** |
| 扫描速度 | 不要求 | **≥ 1°/s（实时 NTM 追踪）** |
| 耐久性 | 不要求抗中子 | **材料需耐中子辐照** |

```

## 11. 联锁保护系统

## 12. 托克马克特有保护功能

### 11.1 等离子体破裂保护

等离子体破裂（Disruption）是托克马克中最严重的异常事件。ECRH 系统必须在破裂时迅速安全关断。

```
破裂保护时序:
═══════════════════════════════════════════════════════════════

  t0: MPS/PCS 检测到破裂前兆
      │
  t0+0: MPS 发出 "破裂预警" 信号 (光纤硬接线)
      │
  t0+100μs: ECRH 系统级安全 PLC 接收信号
      │
  t0+200μs: ECRH-MC 广播 "全通道快速关断" 指令
      │
  t0+500μs: 各通道 FPGA 执行主高压快速关断
      │
  t0+1ms: 所有回旋管微波输出停止 ← 必须在此之前完成
      │
  t0+1ms~: Launcher 开始向安全位置运动
      │
  t0+100ms: Launcher 到达安全位置

  ★ 要求：从 MPS 预警到 ECRH 全部关断 < 1 ms
  ★ 原因：破裂时等离子体消失，无负载的微波束会损伤第一壁
═══════════════════════════════════════════════════════════════
```

### 11.2 VDE 保护

VDE（Vertical Displacement Event）期间，真空室内的 EC Launcher 会承受巨大的电磁力。

| 保护项 | 措施 |
|-------|------|
| Launcher 结构强度 | 结构设计需承受 VDE 电磁力（根据装置参数计算） |
| 驱动机构保护 | VDE 时锁定驱动机构（防止力传递损坏齿轮/电机） |
| 冷却管路 | 柔性连接，吸收变形 |
| 事后检查 | VDE 后自动执行 Launcher 功能自检 |

### 11.3 逃逸电子应对

| 功能 | 说明 |
|------|------|
| 逃逸电子检测接口 | 接收 MPS 的逃逸电子信号（硬X射线探测器/同步辐射探测器） |
| 防护关断 | 当逃逸电子束威胁 Launcher 时，确保 ECRH 已关断 |
| ★ 逃逸电子抑制辅助 | 未来可探索用 ECRH 散射/加热逃逸电子群体，降低其能量（需专门实验验证） |

### 11.4 NTM 抑制控制详细设计

NTM 抑制是 BEST ECRH 系统最重要的先进控制功能：

| 参数 | 要求 |
|------|------|
| 支持的 NTM 模式 | m/n = 3/2 和 2/1 |
| 磁岛检测 | ECE + Mirnov 线圈联合检测 |
| 磁岛宽度测量精度 | ≤ 1 cm |
| 微波沉积位置精度 | Δρ_pol < 0.05（约 2 cm） |
| 响应延迟（检测→执行） | < 5 ms |
| ECCD 效率优化 | 在线射线追踪计算，优化注入角度 |
| 功率调制方式 | 可选连续 ECCD 或与磁岛相位同步的脉冲 ECCD |

## 13. 杂散辐射监测与保护

### 12.1 背景

参考 W7-X 经验，O2 模式下即使多程吸收率达 95%，仍有 5% 的微波功率（10 MW 下约 500 kW）以杂散辐射形式散布。在 BEST 中，杂散辐射问题因以下因素可能更加严峻：

- 托克马克真空室内部结构复杂（包层、偏滤器等增加反射面）
- 包层系统对微波杂散辐射敏感
- 长脉冲运行下杂散辐射热积累效应

### 12.2 监测方案

```
杂散辐射监测系统:
├── 真空室内微波探测器（分布式，≥ 20 个探测点）
│   ├── 类型：肖特基二极管 / 微波功率计
│   ├── 频率范围：覆盖 140 GHz 及其倍频
│   ├── 响应时间：< 1 ms
│   └── ★ 需抗辐射型探测器（真空室内核辐射环境）
├── ★ 包层温度异常监测（杂散辐射引起的局部过热）
├── 端口处微波泄漏监测
│   └── 各诊断/泵送端口处安装探测器
└── 传输线段杂散监测
    └── 传输线沿途关键节点
```

### 12.3 杂散辐射保护策略

| 杂散等级 | 条件 | 动作 |
|---------|------|------|
| 正常 | 所有探测器 < 阈值的 70% | 正常运行 |
| 预警 | 任一探测器 > 阈值的 70% | 操作员报警 |
| 降功率 | 任一探测器 > 阈值 | 自动降功率至杂散恢复正常 |
| 关断 | 多个探测器同时超阈值 | 全系统快速关断 |

| **VDE 信号** | MPS→ECRH | **硬接线（光纤）** | **< 100 μs** | 触发关断，保护 Launcher |
| **逃逸电子信号** | MPS→ECRH | 硬接线/数字 | < 1 ms | 触发关断或切换至抑制模式 |
| **等离子体状态** | MPS→ECRH | 数字 | < 1 ms | 等离子体存在/消失 |
| ECRH 系统状态 | ECRH→MPS | 数字 | < 10 ms | 反馈 ECRH 系统健康状态 |
| ECRH 注入功率 | ECRH→MPS | 模拟/数字 | < 10 ms | MPS 评估总加热功率 |

### 13.6 物理穿越件设计要求

### 14.1 杂散磁场管理

| 区域 | 杂散磁场要求 | 管理措施 |

| 区域 | 辐射来源 | 防护措施 |
|------|---------|---------|

### 14.3 电磁兼容（EMC）设计

托克马克环境存在强电磁干扰（破裂时的瞬态电磁脉冲、大功率射频加热系统等）：

| EMC 措施 | 说明 |
|---------|------|
| 信号电缆屏蔽 | 所有信号电缆使用双层屏蔽，单端接地 |
| 光纤传输 | 安全联锁信号和远距离通信采用光纤，天然抗 EMI |
| 接地系统 | ECRH 系统独立接地网，单点与装置主接地网连接 |
| 控制柜屏蔽 | 近机控制柜采用 EMC 等级屏蔽机箱 |
| 电源滤波 | 控制电源入口加装 EMI 滤波器 |
| 破裂瞬态保护 | 近机电子设备加装瞬态抑制器（TVS） |

### 14.4 传输线区域环境控制

参考 W7-X 教训（2018 年传输线打火事故），传输线走廊环境控制：

### 16.2 时序同步方案

- **全局同步**：IEEE 1588 PTPv2，精度 < 1 μs
- **触发同步**：光纤硬件触发线，精度 < 100 ns
- **脉冲编号**：每次放电统一脉冲编号（与 BEST CCS 同步）

T+30min+2s: 功率斜坡下降

            系统回到 READY 状态，等待下次放电

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 13 | **30 min 稳态运行热管理** | W7-X 经验 | 全系统水冷，温度趋势预测保护 |
| 14 | **★ 端口空间竞争** | BEST 特有 | 赤道面+上端口组合，优化束线密度 |

### 19.2 降额运行策略

参考 W7-X 运行经验（8 MW 降额至 6 MW 以提高可靠性）：

| 情况 | 降额策略 |
|------|---------|
| 环境湿度偏高 (20%-30%) | 各通道最大功率限制至 80% |
| 单台回旋管故障 | 其余通道自动增功率补偿（如不超限） |
| 冷却能力下降 | 按冷却裕度比例降额 |
| 杂散辐射偏高 | 优先降低吸收差的通道功率 |
| 长脉冲 (> 10 min) | 可选降额至 80% 延长寿命 |
| **★ BEST 磁体杂散场偏高** | **评估影响，必要时减少同时运行的回旋管数** |
| **★ 冷屏热负荷超标** | **降低传输功率或减少束线数** |
| **★ 金刚石窗辐照老化** | **降额运行，安排换窗** |
